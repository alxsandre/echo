# Echo

> **Work in progress** — experimental project built through vibe coding with [Claude Code](https://claude.ai/code).

Echo is a minimalist sound-sharing platform. The idea: record everyday sounds, organize them into sonic albums, and discover them through a social feed. Built with a nostalgic, emotionally resonant aesthetic — designed for deep listening.

## Features (in progress)

- **My Space** — personal library of sounds and albums
- **Feed** — discover sounds published by the community
- **Sonic albums** — group sounds into curated collections
- **Audio player** — waveform visualization, playback progress, inline listening

## Stack

- [Next.js 16](https://nextjs.org/) (App Router)
- [React 19](https://react.dev/) with Server Components
- [TypeScript](https://www.typescriptlang.org/) strict mode
- [Tailwind CSS v4](https://tailwindcss.com/)

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
